display 속성

inline : span, b, img, a, i, button
다른 요소들과 같은 줄에 머무르려고 하는 성향
가로 길이는 필요한 만큼만 차지하는 성향

block : div, h, p, nav, ul, li
새로운 줄에 가려고 하는 성향
가로 길이를 최대한 많이 차지하려고 하는 성향

inline-block
inline 요소도 width, height를 지정할 수 있음

Hello my name is young! - i태그 : inline

Hello my
name
is young! - div태그 : block
